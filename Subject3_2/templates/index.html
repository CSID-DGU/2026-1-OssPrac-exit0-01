Open Source Software Project Team

우리 팀은 Flask와 Nginx 기반의 웹 개발 환경을 활용하여 오픈소스소프트웨어 프로젝트 팀을 소개하는 웹 페이지를 제작했습니다. 프론트엔드, 백엔드, 배포 역할을 나누어 협업하며, Python, Flask, Bootstrap, Nginx와 같은 오픈소스 기술을 실제 웹 서비스 구조에 적용했습니다.

Project Goal

팀원의 역할, 기술 스택, 프로젝트 기여도, 포트폴리오와 연락처를 한눈에 확인할 수 있는 팀 소개 웹 페이지를 구현합니다.
